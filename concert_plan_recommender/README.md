# Concert Recommendation Plan

A concert recommendation web application built with django, ML, and exteranl APIs

## Features
- Ticketmaster API integration for concert information
- Mapbox API integration for calculating the travel distance
- OpenAI API integration for generating concert travel plan
- KMeans user segmentation
- Logistic Regression for preference prediction

## Tech stack
- JavaScript
- HTML
- Python
- Tailwind CSS
- scikit-learn
